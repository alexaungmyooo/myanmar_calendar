# Changelog

## [1.0.0] - 2024-11-14
### Added
- Initial release with support for:
  - English (UK)
  - Myanmar Unicode
  - Zawgyi-One
  - Mon
  - Shan (Tai)
  - S'gaw Karen
- Comprehensive date conversion logic from Gregorian to Myanmar calendar.
- Unit tests for date conversion.
- Example application demonstrating package usage.
